<h1> Binary Search Tree in Java </h1> 

This is a simple implementation of constructing a binary search tree in java. <br>
This tree is created and then displayed using the 3 tree traversal techniques. <br>
1. Pre-Order <br> 
2. In-order <br> 
3. Post-order <br>
